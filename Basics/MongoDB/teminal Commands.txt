mongo -> opens mongo shell'

//Some MongoDb commands:
* show dbs -> show all databases

* use "dbname" -> cd to desired Database

* show collections -> show all tables(collections) in current database

* db.collectionName.find() -> shows all values of table(collections)

* db.dropDatabase() -> delete this database
